## 2024-05-24 - [O(1) Data Visualization Lookups]
**Learning:** Found array iterators (`indexOf` and `Object.entries().forEach`) being actively recreated and processed within React functional component render loops (`useMemo` aggregators) for heavy data visualizations (`WorldMap`, `ComponentMatrix`). While functional iterators are declarative, running O(N) operations repeatedly inside render loops blocking the main thread scales poorly as the dashboard dataset grows.
**Action:** Lift constant mapping objects (like severity scales and geography-to-region conversions) OUTSIDE of React component declarations completely. Pre-compute and invert dictionaries on module initialization to guarantee pure O(1) hash map lookups during component renders, preventing unnecessary memory allocation and redundant iterations per render.
